# AgentSDK

This repository contains the SDK for interacting with the Agent API.

## Python SDK

The Python SDK is located in the `PythonSDK` directory.

### Installation

```bash
pip install -e PythonSDK
```

### Development

```bash
cd PythonSDK
pip install -e ".[dev]"
```

## .NET SDK

The .NET SDK is located in the `DotNetSDK` directory.

### Build

```bash
cd DotNetSDK
dotnet build
```
